\begin{frame}[t, fragile]{``Declaring'' a function in Haskell.}

A Haskell function is essentially ``declared'' in a manner which is
illustrated by the following example;

\begin{minted}[linenos]{haskell}
addTwoNumbers :: Int -> Int -> Int
\end{minted}

However, it is worth noting that Haskell doesn't refer to such a statement
as a function declaration. Instead, it refers to it as a type signature,
or a function type signature. \\

This function type signature tells us that the function takes two Ints
as arguments, and returns an Int as the result.

\end{frame}
