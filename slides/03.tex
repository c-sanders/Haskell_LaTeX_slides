\begin{frame}[t, fragile]{Haskell functions.}

According to ChatGPT, in Haskell, every function technically returns a value.
The value returned can be of type (), i.e. the unit value, but the
function is still returning a value. \\

The unit value is considered special in Haskell. It can be thought of as
being similar to the void type in C/C++. \\

Haskell can be thought of as being an expression-oriented language. Every
expression has a value, and a function body is an expression.

\end{frame}
