\begin{frame}[t, fragile]{A simple .cabal file for building an execurable.}

Not only can cabal be used to build binary executables, but it can also
be used to execute them once they have been built. With respect to our
basicExample binary executable, we could use cabal to invoke it as follows;

\begin{minted}[frame=single]{console}
$ cabal run basicExample
\end{minted}

\end{frame}
