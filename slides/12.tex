\begin{frame}[t, fragile]{Haskell libraries, modules, and packages.}

In Haskell, source code can be organised into modules, libraries, and packages. \\

\textbullet{} Modules

Modules are simply individual Haskell source code files; that is, files whose
filename extension is .hs. \\

\textbullet{} Libraries

Libraries are logical collections of code. \\

\textbullet{} Packages

Packages are the units of distribution. \\

\end{frame}
