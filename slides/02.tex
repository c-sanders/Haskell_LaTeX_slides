\begin{frame}[t, fragile]{Defining a function in Haskell.}

To define the previous function, its body is placed immediately after
the function type signature.

\begin{minted}[linenos]{haskell}
addTwoNumbers :: Int -> Int -> Int
addTwoNumbers arg1 arg2 =

    arg1 + arg2
